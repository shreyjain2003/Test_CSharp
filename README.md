# C# Assessment Projects

This repository contains two separate C# console applications located in their respective folders. 

## 📂 Project 1: MediSure Clinic Billing

**Location:** `MediSure_Clinic_Bill`

### Key Features
* **Bill Generation:** Captures Patient Name, Bill ID, and charges for Consultation, Labs, and Medicines.
* **Insurance Logic:** Automatically detects if a patient is insured and applies a 10% discount to the gross amount.
* **View & Clear:** Users can view the most recently generated bill or clear it from the system memory.

---

## 📂 Project 2: QuickMart Traders Profit Calculator

**Location:** `QuickMart_ProfitCalculator`

### Key Features
* **Profit/Loss Analysis:** Takes the Purchase Amount and Selling Amount to determine if the transaction resulted in a Profit, Loss, or Break-Even status.
* **Margin Calculation:** Computes the exact profit or loss amount and the profit margin percentage.
* **Re-calculation:** Includes a feature to re-process and view the logic for the last entered transaction.
